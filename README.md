# Redirector

[![Build Status](https://travis-ci.org/Aehmlo/redirector.svg)](https://travis-ci.org/Aehmlo/redirector) [![License](https://img.shields.io/github/license/aehmlo/redirector.svg)]() [![GitHub issues](https://img.shields.io/github/issues/aehmlo/redirector.svg)]()

A super-simple iOS tweak that (as of now) just redirects the calendar app icon to Fantastical. In the future, it will allow the user to set up redirects to their heart's content in the Settings app.
